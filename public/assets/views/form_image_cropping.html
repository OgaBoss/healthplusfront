IMAGE CROPPING

ct-crop directive

The ct-crop directive makes you easily crop images. Thanks to the powerful Jcrop and Angular Awesome Slider functionality, ct-crop will make you work less and get the best result you can. 

{{cropItem | json}}

To activate the image crop, just use the following code in your html:

<ct-crop url-image="url" crop-options="options" crop-labels="labels" crop-callback="updateCrop"></ct-crop>

The crop-options attribute refers to an object in order set up the crop parameters.

  • url: image directory
  • cropX: the X coordinate of the left top corner from which che crop rectangle starts (default: 0)
  • cropY: the Y coordinate of the left top corner from which che crop rectangle starts (default: 0)
  • cropWidth: the crop rectangle width (measured in pixels)
  • cropHeight: the crop rectangle height (measured in pixels)
  • cropMaxWidth: the crop rectangle maximum width
  • cropMaxHeight: the crop rectangle maximum height
  • cropMinWidth: the crop rectangle minimum width
  • cropMinHeight: the crop rectangle minimum height
  • autoOpenModal: true/false value which specifies whether the crop management modal has to open automatically or not when changing the image url

The crop-labels attribute refers to an object in order customize the modal title, the texts and the button icons.

  • modalTitle: the crop management modal
  • modalButton: an object with two parameters:
    • label: open-modal button text
    • icon: the chosen icon class
  • cropButton: an object with two parameters:
    • label: crop button text
    • icon: the chosen icon class
  • cancelButton: an object with two parameters:
    • label: cancel button tooltip
    • icon: the chosen icon class for Cancel button
  • fitButton: an object with two parameters:
    • label: text appearing in the button tooltips. It eases the image resize according to the cropping area minimum width.
    • icon: the chosen icon class for Cancel button

The crop-callback attribute refers to a function you can activate by clicking on the crop button. It shows an event with all the useful cropping informations.

Image Crop example

Add the image crop directive <img-crop> to the HTML file where you want to use an image crop control.
Note: a container, you place the directive to, should have some pre-defined size (absolute or relative to its parent). That's required, because the image crop control fits the size of its container.
